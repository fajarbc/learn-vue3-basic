# vue-crash-2021
Just an example project for learning Basic of Vue3

## Project setup
1. Install dependency
    ```
    npm install
    ```
2. Copy .env to .env.local
    ```
    cp .env .env.local
    ```

## Run
1. Backend
    ```
    npm run backend
    ```

2. Frontend
    ### Compiles and hot-reloads for development
    ```
    npm run serve
    ```

    ### Compiles and minifies for production
    ```
    npm run build
    ```
